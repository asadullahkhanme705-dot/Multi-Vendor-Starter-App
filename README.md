# 🛒 Multi-Vendor E-commerce Platform

A full-stack multi-vendor marketplace built using the MERN stack (MongoDB, Express, React, Node.js).  
This application allows multiple vendors to manage their products while customers can browse, add to cart, and place orders across different sellers in a unified platform.

## ✨ Features

- 🔐 JWT Authentication (Customer & Vendor roles)
- 🏪 Vendor product management
- 🛒 Cart system with multi-vendor support
- 📦 Order creation & vendor-based order splitting
- 📊 Vendor-specific order dashboard

## 🛠️ Tech Stack

- Frontend: React, Redux
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)

## 🚀 Goal

To simulate real-world marketplaces like Amazon and Daraz with scalable architecture and clean code practices.
